# Schaefer-200 Atlas 脑区功能分区报告

## 1. 概述

Schaefer-2018 atlas 是由 Laurens Maisog 等人于 2018 年发布的基于功能磁共振成像(fMRI)的脑区分割图谱。该图谱通过结合局部梯度分析和功能连接聚类，将大脑皮层分割为 200 个感兴趣区域(ROIs)。

### 1.1 基本信息

| 属性 | 描述 |
|------|------|
| **版本** | Schaefer 2018 |
| **ROIs 数量** | 200 (或 100, 400, 600, 800, 1000) |
| **参考空间** | MNI152 2mm/1mm |
| **分割依据** | 功能连接 + 局部梯度 |
| **原始文献** | "Local-Global Parcellation of the Human Cerebral Cortex" |

### 1.2 网络划分

Schaefer-200 默认将大脑划分为 **7 个功能网络**：

| 网络编号 | 网络名称 | 英文名 | 颜色代码 | 主要功能 |
|---------|---------|--------|----------|---------|
| 1 | 视觉网络 | Visual (Vis) | 蓝 | 初级视觉加工 |
| 2 | 躯体感觉网络 | SomatoMotor (SomMot) | 橙 | 躯体感觉和运动 |
| 3 | 背侧注意网络 | Dorsal Attention (DorsAttn) | 浅绿 | 空间注意和眼动 |
| 4 | 腹侧注意网络 | Ventral Attention (VentAttn) | 深绿 | 刺激驱动注意 |
| 5 | 边缘系统 | Limbic | 黄 | 情绪和记忆 |
| 6 | 额顶控制网络 | Frontal Parietal (FpnCont) | 红 | 执行控制和认知 |
| 7 | 默认模式网络 | Default Mode (Default) | 紫 | 内省和社会认知 |

## 2. 详细 ROI 序列映射表

### 2.1 网络 1: 视觉网络 (Visual Network) - ROI 1-28

视觉网络主要位于大脑后部的枕叶区域，负责处理视觉信息。

| ROI 序号 | 半球 | 皮层区域 | 功能描述 |
|----------|------|----------|---------|
| 1-7 | 左 | V1 (初级视皮层) | 原始视觉信息接收 |
| 1 | L | Calcarine (距状沟) | 初级视觉皮层，视野映射 |
| 2 | L | Cuneus (楔叶) | 上视野初级视觉 |
| 3 | L | Lingual (舌回) | 下视野初级视觉 |
| 4 | L | Pericalcarine | 视辐射终点 |
| 5-7 | L | 高阶视觉区 | V2, V3, V4 |
| 8-14 | 右 | V1 (镜像) | 右侧初级视觉皮层 |
| 8 | R | Calcarine | 右侧距状沟 |
| 9 | R | Cuneus | 右侧楔叶 |
| 10 | R | Lingual | 右侧舌回 |
| 11-14 | R | 高阶视觉区 | V2-V4 |
| 15-21 | 左 | 视觉关联区 | 外侧枕叶复合体(LOC) |
| 22-28 | 右 | 视觉关联区 | MT+, 视觉运动区 |

**功能注释**：
- ROI 1-3: 初级视觉区(V1)，处理视野中与视网膜对应位置的视觉输入
- ROI 4-7: V2/V3，边缘和轮廓检测
- ROI 8-14: 右侧对应区域
- ROI 15-21: 腹侧视觉通路，处理物体识别
- ROI 22-28: 背侧视觉通路，处理空间位置和运动

### 2.2 网络 2: 躯体感觉-运动网络 (SomatoMotor Network) - ROI 29-56

该网络涵盖中央前回(运动皮层)和中央后回(感觉皮层)。

| ROI 序号 | 半球 | 皮层区域 | 功能描述 |
|----------|------|----------|---------|
| 29-35 | 左 | 中央前回/后回 | 运动和感觉表征 |
| 29 | L | Precentral (中央前回) | 运动皮层，手部表征 |
| 30 | L | Postcentral (中央后回) | 体感皮层 |
| 31 | L | Paracentral Lobule | 足部运动/感觉区 |
| 32 | L | Rolandic Operculum | 面部/吞咽区 |
| 33-35 | L | 中央旁小叶 | 躯干表征 |
| 36-42 | 右 | 中央前回/后回(镜像) | 对侧躯体表征 |
| 43-49 | 左 | 辅助运动区 | SMA, Pre-SMA |
| 43 | L | SMA | 运动序列规划 |
| 44 | L | Pre-SMA | 运动选择 |
| 45-49 | L | 上运动区 | 精细运动控制 |
| 50-56 | 右 | 辅助运动区(镜像) | 对侧辅助运动 |

**功能注释**：
- **ROI 29-30**: 经典的"皮层小人"区域，嘴-手-足的运动/感觉映射
- **ROI 32**: Rolandic operculum 涉及面部表情和语言产生
- **ROI 43-44**: 辅助运动区(SMA)负责运动序列的内部规划

| 59 | L | IPS2 | 注意切换 |
| 60-63 | L | IPS3-5 | 后部顶叶 |
| 64-70 | 右 | 顶上小叶(镜像) | 对侧空间注意 |

| 76-78 | 右 | 额眼区(镜像) | 对侧眼动控制 |

**功能注释**：
- **ROI 57-63**: 顶内沟(IPS)是背侧注意网络的核心，涉及空间注意的定向

| 81 | L | Supramarginal Gyrus | 体感整合 |
| 82 | L | MTL边缘 | 记忆相关注意 |
| 83-86 | 右 | TPJ(镜像) | 对侧显著刺激检测 |

| 91-92 | 右 | 腹侧额叶(镜像) | 对侧任务控制 |

**功能注释**：
- **ROI 79-82**: 颞顶交界处(TPJ)被称为"警觉中心"，检测意外刺激
- **ROI 87-90**: 腹侧额叶皮层，尤其是 IFG 和前岛叶，负责抑制优势反应

**功能注释**：

- **ROI 99-100**: 前扣带回膝部(subgenual ACC)与情绪调节密切相关

## 5. 在本项目中的应用

### 5.1 数据输入格式

每个 ROI 对应一个 200 维的特征向量：

```python
# 输入形状: (batch_size, n_rois=200, n_features)
# n_features: [GM_mean, GM_volume, GM_max_intensity]
```

### 5.2 特征解释

| 特征索引 | 特征名称 | 单位 | 来源 |
|----------|----------|------|------|
| 0 | GM_mean | 标准化强度值 | 灰质概率均值 |
| 1 | GM_volume | mm³ | 灰质体积 |
| 2 | GM_max_intensity | 标准化强度值 | 峰值强度 |

### 5.3 ROI 序列与脑区对应

在本项目的模型中，输入的 ROI 序列严格按照 Schaefer-200 的原始顺序排列：

```python
# ROI 索引到脑区映射 (简化)
roi_to_region = {
    0-27: "Visual Network (双侧枕叶)",
    28-55: "SomatoMotor Network (双侧中央前后回)",
    56-77: "Dorsal Attention Network (双侧顶内沟)",
    78-91: "Ventral Attention Network (双侧颞顶交界)",
    92-99: "Limbic System (颞叶内侧)",
    100-141: "Fronto-Parietal Control (双侧前额叶)",
    142-199: "Default Mode Network (内侧皮层)"
}
```
